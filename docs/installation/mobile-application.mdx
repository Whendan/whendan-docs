# Mobile Application


The Whendan mobile application is designed to provide drivers with a seamless experience while on the road. 
It offers features such as real-time dispatch notifications, route optimization and easy access to delivery details. 
The app is available for Android devices, ensuring that drivers can stay connected and informed no matter where they are. 
With the mobile application, drivers can efficiently manage their deliveries and stay updated on any changes or updates from the dispatch team.

## Downloading the App

The mobile application is currently in development and will be available for download on the Google Play Store soon.
In the meantime, if you are interested in testing the app or want to learn more, it is available on GitHub at https://github.com/Whendan/whendan-mobile-app

## Onboarding
To get started with the Whendan mobile application on a self hosted instance, drivers will need to scan the QR code provided by the dispatch team available at `/dashboard/settings/mobile`
