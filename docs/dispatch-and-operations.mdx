# Dispatch and Operations

WhenDan supports dispatch-related operations through package assignment, driver workflows, and route data.

## Operational Capabilities

- Package assignment and tracking workflows
- Driver and vehicle assignment support
- Driver shift and route progression visibility
- Warehouse-centered dispatch coordination

## Optimization Context

Route and optimization-related models support planning and analysis.
These models can be used to improve assignment quality and reduce waste in daily operations.

## Language Guidance

Current docs use the phrase "dispatch support" intentionally.
As additional automation surfaces are finalised, this section will document which dispatch actions are manual, assisted, or fully automated.

## Outcome Focus

The operational goal is consistent:

- Higher equipment utilization
- Fewer late deliveries
- Lower fuel and labor waste
- Better visibility for dispatch and service teams
