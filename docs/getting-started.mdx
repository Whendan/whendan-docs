# Getting Started

This guide helps you run WhenDan locally and understand the project layout.

## Prerequisites

- Node.js 20+
- pnpm
- Supabase project and credentials

## Run the App

From the main app workspace:

1. Copy `.env.example` to `.env.local`.
2. Fill required environment variables.
3. Install dependencies.
4. Start development server.

```bash
pnpm install
pnpm dev
```

App URL:

- http://localhost:3000

## Core App Areas

- Auth flows: login, password reset, account recovery
- Dashboard: packages, customers, fleet, driver shifts, service workflows
- Data layer: Supabase clients, RPC wrappers, typed database models
- Mapping and routing: OpenRouteService integration and map components

## Recommended First Reads

- Main project overview in the app README

- Supabase typed schema and RPC wrappers in lib/supabase
